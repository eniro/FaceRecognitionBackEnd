<h2>Description</h2>

React app that utilizes Clarifaiâ€™s facial recognition API and detects faces in photos. There is also a register/login page that saves user information to a database created with PostgreSQL in order to count user's entries (how many times a user has used a photo in the app).  
Languages used for the front-end: React.JS, Javascript, HTML5, CSS3.<br>
Languages used for the back-end: Node.js, Express.js, SQL(PostgreSQL).<br>
Background was created with Particles.js library.
